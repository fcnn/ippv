IPPV
======================================================
  International phone-number parser and validater. It
can be used to check if a phone number starts with a
valid country code, aka, country dailing code.


usage
-------------------------------------------------------
  - c/c++:
    simply use phone_no.h and phone_no.c in your projects
  - kotlin:
    use PhoneNo.kt
  - java:
    use PhoneNo.java
