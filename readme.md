IPPV
======================================================
  internal phone-number parser and validater. it
can be used to check if a phone number starts with a
valid country code, aka, country dailing code.


usage
-------------------------------------------------------
  - c/c++
    simply use phonenr.h and phonenr.c in your projects
  - kotlin
    use PhoneNr.kt
  - java
    use PhoneNr.java
