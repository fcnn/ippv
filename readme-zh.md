IPPV
======================================================
    校验/解析国际电话号码格式。用于检查电话号码是否以有
效国家拨号码(country dialing code，或简称 国家码：
country code)开头。



使用说明
-------------------------------------------------------
- c/c++
   使用phonenr.h和phonenr.c这2个文件即可。

- kotlin
   使用PhoneNr.kt

- java
   使用PhoneNr.java
